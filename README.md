# insta_info
This program scraps/extracts public account information on Instagram

Copyright:
Instagram: @a7.acc
Telegram: @A7_acc

Grabs an Instragram account public data

The information it extracts:
- UserID
- Name
- Username
- Followers
- Following
- Posts
- Bio URL
- Is private
- Is verified
- Highlight reel count
- Is professional account
- Is business account 
- Linked Email
- Business email
- Business phone number
- User Create Time
- Account Region
- Bio
